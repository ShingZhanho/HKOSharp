OneDayWeather Class

A class that can deserialize receiven JSON from API for a day's forecast.


                public class OneDayForecast

Constructors

OneDayWeather(string)

(internal) Instantizes a new OneDayWeather instance to the specific JSON string.

emoji_objects Note

You cannot instantize a new OneDayWeather instance in your code since all its constructors are internal. OneDayWeather objects are included in NineDaysWeather objects.

Fields

ForecastDate DateTime

Represents which day is this forecast for.
Week string

Represents the day of a week (Sunday, Monday, Friday etc.). This field's language depends on the Language parameter of methods GetNineDaysWeather() or GetNineDaysWeatherAsync().
ForecastWind string

Represents the wind forecast of the day.
ForecastWeather string

Decription of weather forecast of the day.
ForecastMaxTemp double

Represents the predicted highest temperature of the day (in degrees Celsius).
ForecastMinTemp double

Represents the predicted lowest temperature of the day (in degrees Celsius).
ForecastMaxRh double

Represents the predicted highest relative humidity of the day (in percent).
ForecastMinRh double

Represents the predicted lowest relative humidity of the day (in percent).
ForecastIcon int

Represents the number of a forecast icon. Check Hong Kong Observatory's website for the full list of weather icons.
