SeaTemp Class

A class that can deserialize received JSON from API for sea temperature.


                public class SeaTemp

Constructors

(internal) Instantizes a new SeaTemp instance to the JSON object.
SeaTemp()

emoji_objects Note

You cannot instantize a new SeaTemp instance in your code since all its constructors are internal. SeaTemp objects are included in NineDaysWeather objects.

Represents the name of place where the sea temperature is measured.

Represents the temperture of sea measured (in degrees Celsius).
