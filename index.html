What is HKOSharp?

HKOSharp is an unofficial C# Library of Hong Kong Observatory's API. It handles all the HTTP requests inside the library. HKOSharp allows you to access Hong Kong's Weather Data with one line or two.

Here is an example showing how to get the tomorrow's forecast:


                    var forecasts = LibHKOSharp.Weather.GetNineDaysWeather(Language.English);
                    Console.WriteLine(forecasts.WeatherForecast[0].ToString());

Go to docs page to learn more.
